How a request is being processed ...

  • The Yanel servlet receives a request (e.g. /yanel/en/documentation/how-a-request-is-being-processed.html) and is dispatching it within the service method
  • The resource type is being determined by looking up REQUEST.yanel-rti (e.g. /yanel/en/documentation/how-a-request-is-being-processed.html.yanel-rti)
  • The determined resource type (e.g. <{http://www.wyona.org/yanel/resource/1.0}xml/>) is being loaded dynamically and is processing the request resp. generates the content
