About

Everything is content management (WCMS/Wiki, DMS, CRM, ERP, PIM/E-Mail, DRM, ...) and Yanel is a humble attempt to provide a solution for all your content management needs by offering an Open Source service oriented Content Management Framework written in Java.

The Yanel community has defined a set of principles to sustain long-term development.

Download

See the download page for information on releases, snapshots, source code, etc.

Main Features

  • Resource Framework with versioned published interfaces which allow continuous development and integration
  • Supports the Neutron protocol, Atom Publishing Protocol and WebDAV
  • Clusterable
  • Arbitrary Content Repositories (e.g. File System, Subversion, Relational DB, ...)
  • Separation of Content and Layout
  • Multiple Sites
  • Reverse Proxy Support
  • Cross-Platform (Continuously tested on Linux, Mac OS X, Windows)
