Über Yanel

Alles ist Content Management (WCMS, DMS, CRM, ERP, PIM/E-Mail, DRM, ...) und Yanel ist ein bescheidener Versuch eine Lösung für alle Content Management Bedürfnisse anzubieten durch ein Open Source Service-orientiertes Content Management Framework geschrieben in Java.

Die Yanel community hat eine Reihe von Prinzipien definiert um eine langfristige Entwicklung und einen entsprechenden Investitionsschutz zu garantieren.

Features

  • Resource Framework mit versionierten Schnittstellen das fortlaufende Entwicklung erlaubt
  • Unterstützt das Neutron Protokoll
  • Clusterfähigkeit
  • Beliebige Content Repositories
  • Trennung von Inhalt und Layout
  • Mehrere Sites
  • Reverse Proxy Unterstützung
  • Cross-Platform (Linux, Mac OS X, Windows (ungetestet), ...)
