Ueber Yanel

Alles ist Content Management (WCMS, DMS, CRM, ERP, E-Mail, DRM, ...) und Yanel ist ein bescheidener attempt to provide eine Loesung for all your content management needs by offering an Open Source service oriented Content Management Framework geschrieben in Java.

Die Yanel community has defined a set of principles to sustain long-term development.

Download

See the download page for information on obtaining releases, snapshots, etc.

Features

  • Resource Framework with versioned interfaces which allow continuous development
  • Supports the Neutron protocol
  • Clusterable
  • Arbitrary Content Repositories
  • Separation of Content and Layout
  • Multiple Sites
  • Reverse Proxy Support
  • Cross-Platform (Linux, Mac OS X, Windows (not tested yet), ...)
